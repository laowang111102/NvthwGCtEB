# 社团管理系统 java751

社团管理系统是一个基于前后端分离架构的项目，采用 Spring Boot 和 Vue.js 技术栈，旨在为各类社团提供一个高效便捷的管理平台。

## 技术栈

- **后端：**
  - Java
  - Spring
  - Spring Boot
  - MVC
  - MyBatis
  - MySQL
  - Maven

- **前端：**
  - HTML
  - JavaScript
  - CSS
  - Vue.js

## 运行环境

- **开发工具：**
  - IntelliJ IDEA
  - VS Code

- **运行依赖：**
  - JDK 1.8
  - MySQL 数据库
  - Node.js

## 系统功能模块

系统主要分为以下几个功能模块：

1. 用户管理
2. 社团信息管理
3. 活动发布与管理
4. 资源共享平台
5. 通知公告发布

## 目录结构

```
项目根目录
│
├── 后端源码目录
│   ├── src
│   ├── pom.xml
│   └── ...
├── 前端源码目录
│   ├── src
│   ├── package.json
│   └── ...
└── 数据库文件
    └── schema.sql
```

## 核心亮点

- **前后端分离**：后端提供 RESTful API，前端 Vue.js 进行数据渲染，便于分工开发和维护。
- **高效开发**：基于 Spring Boot，实现快速开发，减少不必要的工作量。
- **易用性**：提供详细的文档和示例代码，降低二次开发难度。

## 部署步骤

1. 安装 JDK 1.8
2. 安装 MySQL 数据库，并导入数据库文件
3. 使用 IntelliJ IDEA 或 VS Code 打开后端项目，编译并运行
4. 使用 Node.js 和 VS Code 打开前端项目，编译并运行

注意：以上内容仅供参考，具体部署步骤请结合实际环境和需求调整。
